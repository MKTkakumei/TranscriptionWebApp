# 文字起こしWebアプリ (Transcription Web App)

リアルタイムで音声認識を行い、文字起こしを行うWebアプリケーションです。
段階的に機能を拡張し、最終的には話者分離機能の実装を目指しています。

## 機能概要

### フェーズ1: 基本機能 (完了)
- **リアルタイム文字起こし**: マイク入力から即座にテキスト化します。
- **タイムスタンプ表示**: 発言ごとの時間を記録・表示します。
- **ログ保持**: ブラウザを開いている間、ログを蓄積します。

### フェーズ2: ログ保存と話者特定 (完了)
- **話者名入力**: 画面上で自分の名前を設定し、ログに記録できます。
- **ログ保存**: 文字起こしログをテキストファイルとしてダウンロードできます。

### フェーズ3: ログ統合機能 (完了)
- **ログ統合**: 複数人のログファイルを読み込み、時系列順に結合して1つの議事録を作成します。
- **ファイル管理**: 重複排除、個別削除、一括クリアなどの管理機能を提供。

### フェーズ4: AI話者分離 (将来的な拡張)
1つのマイクで複数人の声を拾う場合の補助機能として実装。
※フェーズ3まででブラウザ完結の有用なツールとなるため、まずはフェーズ3までの完成を目指します。

- **技術スタック**: `Pyannote.audio` (Python)
- **機能要件**: 声の特徴から話者を識別。

## 技術スタック

- **フロントエンド**: HTML, CSS, JavaScript (Vanilla)
- **STTエンジン**: Web Speech API

## 使い方

### 文字起こし
1. https://mktkakumei.github.io/TranscriptionWebApp/ をChromeブラウザで開きます。
2. 「話者名」を入力します。
3. 「文字起こし開始」ボタンを押してマイクの使用を許可します。
4. 話しかけると、画面に文字起こし結果が表示されます。
5. 会議終了後、「ログ保存」ボタンを押してテキストファイルをダウンロードします。

### ログ統合
1. 画面中央の「ログ統合機能」をクリックして展開します。
2. 「ファイルを追加」ボタンから、統合したいログファイル（テキスト形式）を複数選択します。
3. リストを確認し、不要なファイルがあれば「×」ボタンで削除します。
4. 「統合して保存」ボタンを押すと、時系列順に結合されたログファイルがダウンロードされます。
